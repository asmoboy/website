The methods

Seven checks on every batch — tested by Janoshik

Reverse-Phase HPLC

Purity quantified against reference standards — Janoshik’s core test, and the headline number on every certificate.

Mass Spectrometry (LC-MS)

Molecular identity confirmed by mass-to-charge analysis — the compound is what the label claims, not a substitute.

TFA Content

Residual trifluoroacetic acid measured — relevant for peptide synthesis byproducts that HPLC purity alone won’t flag.

Endotoxin (LAL)

Bacterial endotoxin levels screened against a defined threshold.

Sterility

Checked separately from purity — a chemically pure batch can still carry bacterial contamination, so it’s tested independently.

Heavy Metals

Trace elemental contaminants screened on request for batches where it matters.

Batch-Level Archival

Each batch carries a unique Lot ID linked to its Janoshik COA — verifiable directly on janoshik.com, exact paperwork for the exact vial.
